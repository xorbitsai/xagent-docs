### Xagent v0.6.2 — July 17, 2026

ElevenLabs voice cloning, a visual Flow view for Custom Agents, and MCP OAuth dynamic registration all land.

#### How to update

```bash
cd xagent && git pull
docker compose up -d
# or
pip install --upgrade xagent-ai
```

**Highlights**
- 🎙️ ElevenLabs voice cloning & audio previews — clone voices and preview generated audio directly in the app
- 🧩 Custom Agent Flow view — the agent builder adds a visual Flow view for orchestrating execution
- 🔐 MCP OAuth dynamic client registration — supports the OAuth dynamic client registration flow for MCP

**What's New**
- 🔑 OAuth resolver runtime refresh — expired authorizations can be renewed without restarting the service
- 📎 v1 SDK task attachments — task turns on the v1 SDK now support file attachments
- 🔄 Durable task command transport — improves task execution reliability
- 📄 Doc Summarizer & Action Extractor template — a ready-to-use P0 template
- 🏷️ Localized built-in model indicator — supports localized copy and badges
- 📊 Token usage by model — breakdowns by individual model
- 🎙️ ElevenLabs voice cloning & audio previews
- 👀 Admin read-only view of agent config
- 🔐 MCP OAuth dynamic client registration
- 👥 Team-scoped agent ownership & visibility
- 🧭 Connector runtime scoped by task
- 🎨 Revamped templates/tools pages & sidebar
- 🧩 Flow view for Custom Agent builder
- 🏢 Provider details across model selectors
- 📥 Import agents from openclaw hermes
- 🔗 Opt-in team ownership via hooks
- 📝 Doc Summarizer template update
- 🧠 Scope-dimension reap & enumeration for memory
- ⚡ Routed agent context optimization
- 🔊 Improved ElevenLabs pronunciation & interruption
- 🗑️ Removed Google image search tool
- 📁 Normalized file references

**Stability & Polish**
- Normalize DeepSeek tool protocol responses
- Always show a thin scrollbar instead of revealing it on hover
- Install HTTPX SOCKS support
- Scope the execution plan to the DAG process
- Avoid an extra DashScope embedding-model list GET request
- Harden DeepSeek tool protocol recovery
- Preserve connector settings during edits
- Use the public API URL in A2A agent cards
- Prevent MCP initialization from exhausting the DB connection pool
- Handle Telegram voice messages and cloning tools
- Surface failed task reasons live, plus a coded-error seam
- Preserve audio playback and pending tool work
- Poll for the detached resume-rejection payload (test)
- Make built-in MCP runtime configuration deterministic
- Make database startup failures self-explanatory
- Fix Gmail trigger OAuth expiry timezone and Pub/Sub provisioning
- Repair workforce setup and execution
- Stop unconfigured agents from inheriting user custom APIs
- Skip malformed rows in LanceDB search instead of truncating results
- Install browser and router runtimes
- Interpret nested metadata filters in InMemoryMemoryStore
- Honor tags/keywords/date filters in LanceDBMemoryStore.search()
- Prevent concurrent auth refresh from causing logout
- Bump compose images to 0.6.1
